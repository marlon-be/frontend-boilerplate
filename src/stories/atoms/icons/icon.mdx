# Icons

## Variable info
Sizes of Icons are exposed in the theme layer, this makes it handy for size calculations in components using icons:

```css
@layer theme {
	:root {
		--icon-size-sm: 0.75rem;
		--icon-size-md: 1.5rem;
		--icon-size-lg: 2.25rem;
		--icon-size-xl: 3rem;
	}
}
```

## Usage

```
<Icon name="download" />
```
